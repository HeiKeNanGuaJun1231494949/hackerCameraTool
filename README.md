# hackerCameraTool
中国黑客组织

Termux安装hackerCameraTool执行命令：

apt-get install python3

apt-get install git

git clone https://github.com/HeiKeNanGuaJun1231494949/hackerCameraTool

pip3 install requests

pip3 install colorama

cd hackerCameraTool

python3 hackerCameraTool.py
